Control iPhone with an external keyboard  
Go to Settings > Accessibility > Keyboards, tap Full Keyboard Access,  
then turn on Full Keyboard Access.  
  
---  
  
* app 전환 (search)  
  
    * alt + space  
    * f1 x 2 times  
  
* Home  
  
	* cmd + h  
  
* Chrome  
  
	* goto address bar  
  
		* cmd + L  
  
	* new tab  
  
		* cmd + t  
  
	* close tab  
  
		* cmd + w  
  
	* switch tab  
  
		* ctrl + tab  
  
	* search  
  
		* cmd + f  
  
	* bookmark  
  
		* opt + cmd + b  
  
	* scroll  
  
        *address bar 또는 content area touch 후 key binding 사용  
  
* Editing  
  
    * 한/영 전환  
  
        * f12  
        * ctrl + space (Chrome에서는 scroll-down, text-field에서는 한/영 전환)  
  
    * Page Up, Page Down  
  
        * fn + up  
        * fn + down  
  
    * Home, End  
  
        * ctrl + up  
        * ctrl + down  
  
    * start / end of line  
  
        * ctrl + a  
        * ctrl + e  
  
    * copy / paste / cut  
  
        * cmd + c  
        * cmd + v  
        * cmd + x  
  
    * jump by word  
  
        * opt + left  
        * opt + right  
  
    * select word  
  
        * fn + right
